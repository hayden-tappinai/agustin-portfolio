◆ SPEC — TICKET RIP-OFF (HERO)
Trigger Click the ticket / tag / "VISITED" stub.
What moves Stub detaches at the seam, falls + rotates + fades; main piece recoils & settles.
Duration ~520ms · token --t-tear (ambient loop runs slower so it reads in the preview).
Easing --ease-thunk on the split · --ease-paper on the fall.
IMPL — CSS (clip-path halves + @keyframes) 
/* two halves of one ticket, shared seam */
.stub{ clip-path:inset(0 68% 0 0);
       transform-origin:right center; }
.main{ clip-path:inset(0 0 0 32%); }

.ticket.torn .stub{
  animation:rip var(--t-tear)
            var(--ease-paper) forwards; }

@keyframes rip{
  0%  { transform:translate(0,0) rotate(0);
        opacity:1; }
  30% { transform:translate(-4px,2px)
        rotate(-3deg); }            /* split */
  100%{ transform:translate(-30px,160px)
        rotate(-30deg); opacity:0; }/* fall */
}
IMPL — Framer Motion (AnimatePresence) 
// stub unmounts on tear -> exit animates
<AnimatePresence>
  {!torn && (
    <motion.div
      style={{ transformOrigin:"right center" }}
      transition={{ type:"spring",
        stiffness:300, damping:25 }}
      exit={{ y:160, rotate:-30,
              opacity:0 }}
    />
  )}
</AnimatePresence>
// main piece: spring back to x:0 on release
WHY — the motif

The ticket is the system's recurring shape: primary button, boarding-pass nav, tags, back-to-globe control. Tearing it is the one interaction everyone remembers.

WHY — paper physics

Spring over linear. Things have mass: the stub drops and settles rather than slides. --ease-thunk gives the split an overshoot; --ease-paper a weighted glide.

WHY — one spark

Volt appears exactly once as the focal accent — the seam flash at the moment of the tear. The travel arrow and stub line are warm gold; everything else stays pitch-green and ink on kraft.
