◆ THE COHESION RULE

The globe stays minimal, dark, and legible — navy ocean, green land, faint graticule, one volt "you-are-here" dot. The bold World-Cup & scrapbook system (bunting, kraft, Polaroids, stamps, scoreboard) lives entirely around that calm panel. Personality wraps the map; the map never fights for attention. This is how every page composes: collage on the edges, clarity at the core.

// WHY THESE CHOICES — drop-in notes for Claude Design
AntonStadium-banner display for the hero headline, brand mark, scoreboard labels & country names — the loud, uppercase signature voice.
Navy panelThe globe sits on globe-navy with a green graticule so the mock matches the real interactive globe exactly — cohesion, not decoration.
Volt ×1The single electric spark is the pulsing you-are-here dot, echoed only as the active nav chip + live-miles glow + ghost-link underline. Never two competing.
Space MonoAll coordinates, gate/seat fields, the barcode label and scoreboard digits — data reads as boarding-pass / scoreboard telemetry.
CaveatHand sub-note & Polaroid captions add the journal warmth without shouting — kept off the structural type.
Kraft + gridWarm paper carries a faint modern grid underneath the collage: tactile texture on a sharp, intentional structure — bold, current, not twee.
OverlapThe destination card breaks the globe panel edge with a torn deckle + rotation so layers feel physically stacked — scrapbook depth.
Zones labeledBlack tags mark each region (masthead → globe → peek → scoreboard → footer) so the page-assembly recipe is explicit.
Higgsfield slotThe hand-drawn soccer-ball fly-through near the globe is a generated transparent WebM — never CSS-coded (SPEC §7/§8). The system leaves a clean labeled slot; the static b/w ball glyph in the Polaroid stays as a flat motif only.
