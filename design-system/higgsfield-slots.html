Where the drawn, hand-animated moments live. The design system reserves the space; it never fakes the motion.

Drawn / character motion is generated in Higgsfield — b/w hand-drawn style, start frame + end frame on green → animated → background stripped to a transparent WebM — then dropped into the React site. We never fake these in CSS; that looks brittle. The design system just reserves a clean, intentional slot.

WHY A SLOT, NOT CODE — HAND-DRAWN MOTION CARRIES THE BRAND. CSS APPROXIMATIONS READ AS CHEAP AND STIFF. A LABELED DASHED FRAME IS HONEST: IT SAYS “A REAL ASSET MOUNTS HERE” AND KEEPS THE LAYOUT INTACT WHILE THE WEBM IS PRODUCED.
